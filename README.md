# Mood Journal
![Project Screenshot](/src/assets/screenshot/main.png)

Минималистичное веб-приложение для ведения дневника настроения.  
Позволяет отслеживать эмоции, делать заметки и сохранять их прямо в браузере.


## Стек технологий
- React  
- TypeScript  
- Vite  
- CSS Modules  


## Основной функционал
- Выбор текущего настроения  
- Ввод и сохранение текстовой заметки  
- Сохранение данных в `localStorage` 
- Минималистичный, чистый интерфейс 

![Project Screenshot](/src/assets/screenshot/choice.png)


## Цель проекта
Практика работы с React, состояниями и UI, создание интерактивного, простого в использовании приложения.


## Структура
- `src/components` — UI-компоненты
- `src/pages` — страницы
- `src/types` — типы TypeScript
- `src/hooks` — пользовательские хуки
